Games

A list of games made with friends or on my own.

Endless Keep

Endless Keep is a rouge like dungeon crawler with limitless levels where the player must defeat waves of enemies to progress to the next level.

WASD Arrow - Hard

WASD Arrow is a five second game where the player has to correctly type the four key combinations to win. The game was originally made to be part of a main game composed of many five second games in a Wario World format. However, this vesion was to difficult and was used in the boss levels instead.

WASD Arrow - Easy

WASD Arrow Two is the second iteration of WASD Arrow; in this version the player only has to correctly type key combinations for two keys instead of four to win.
